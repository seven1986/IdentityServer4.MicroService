    安装¶

    使用OAuthApp可以快速建立OAuth2.0的Saas用户中心。参考如下步骤创建项目并确保已安装Visual Studio 2019、DotNet Core 3.1+、SQL Server。

    1，创建应用程序¶

    2，安装Nuget包¶

    对项目的依赖项点右键，选择管理Nuget程序包

    输入OAuthApp，然后点击安装

    3，配置¶

    3.1 appsetting.json¶

    对项目依次点击鼠标右键——属性——调试，复制项目网址.

    复制下面代码到 appsetting.json 文件，注意将Host节点替换为当前项目网址，结尾不要带“/” 。

    DataBaseConnection 可替换为实际数据库的地址，SMS 和 Email 使用的是SendCloud的服务。

    public void ConfigureServices(IServiceCollection services)
{
    services.AddOAuthApp();
}

public void Configure(IApplicationBuilder app, IWebHostEnvironment env)
{
     if (env.IsDevelopment())
     {
         app.UseDeveloperExceptionPage();
     }
     else
     {
         app.UseExceptionHandler("/Home/Error");
         // The default HSTS value is 30 days. You may want to change this for production scenarios, see https://aka.ms/aspnetcore-hsts.
         app.UseHsts();
     }

     app.UseHttpsRedirection();

    //必须在app.UseHttpsRedirection之后、app.UseStaticFiles之前
     app.UseOAuthAppUI(); 

     app.UseStaticFiles();
}

    4， 启动¶

    启动项目并预览。默认用户账号：admin@admin.com，密码：123456aA!，拥有所有权限。

    5， Swagger UI¶

    在浏览器中打开 {项目网址} /swagger/index.html可访问API文档。

    6，Postman¶

    使用 Postman 测试现有接口。
