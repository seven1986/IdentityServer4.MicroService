    权限码表¶

    OAuthApp内置的接口权限，使用Client请求用户授权时，scope参数可参考下面的code。

    标识权限¶

    code name
    openid 用户的ID
    profile 用户资料详情
    address 联系地址
    email 邮箱地址
    phone 手机号
    role 角色
    permission 权限
    offline_access 脱机访问
