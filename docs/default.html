基于Identityserver4、Azure API Management的开放平台解决方案。

使用Azure API 网关组合多个微服务，以产品包的形式提供使用。

API文档¶

脚手架¶

应包含产品接口SDK的命令行工具。快速生成基于产品的seed项目，减少项目初始化工作。 常见形式：angular cli、vue cli、react cli 等等
